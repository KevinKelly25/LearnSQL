Basic Select Statements

SELECT is a clause used to return data from a database. The data returned is stored in a result table, called the result-set. To use Select in an SQL statement you write SELECT then the name of the attribute columns, each seperated by a comma. SELECT has to be used with the FROM clause, which designates which table the SELECT attributes are coming from. This is shown here:

Example 1

  SELECT attribute1, attribute2, attribute3, ...
  FROM table_name;
            

  Here, column1, column2, ... are the field/attribute names of the table you want to select data from.

If you want to select all the fields available in the table, use * in place of the attributes. As shown in the next example.

Example 2

  SELECT *
  FROM table_name;
            

  Here, all fields in the table are selected because of the use of * in place of the attributes.
