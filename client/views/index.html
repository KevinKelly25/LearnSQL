Learn On Live SQL Database

We take advantage of the fact we have a live PostgreSQL database running in the background. Postgres is a SQL Database Management System that is free and open source. What makes it special is that it uses to the SQL standard for most features. This means that using Postgres to learn SQL is very easy and useful

All of our questions use this Postgres database to preform queries against the current tables of the database. This means that the answers and results are dynamic depending on the current status of the tables. Since we compare results rather then syntax the tables can change all they want but the correct answer will never change
The Workshop uses this live database to provide students with a sandbox environment to do whatever SQL work they want without the worry of effecting anyone else's work. Instructors are also able to monitor students work and provide feedback to the students
